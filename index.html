Numbers Memory Game

HOW TO PLAY

  • This is a turn-based game. Both players will type one of the random numbers between 0 to 9 at each turn
  • The first player types one random number and the next player must remember the previously typed number plus another number of your choice
  • Type the numbers and HIT ENTER!

EXAMPLE (at each turn)

Player1 types 0

Player2 types 09

Player1 types 096

Player2 types 0962
